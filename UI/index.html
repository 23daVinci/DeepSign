Under the Hood

Deep learning architecture designed for one-shot learning.

Siamese Network

Two identical subnetworks process the reference and questioned signatures simultaneously, sharing the exact same weights and parameters.

Feature Embedding

The images are converted into low-dimensional feature vectors (embeddings). The network learns to place similar signatures close together in this space.

Distance Metric

We calculate the Euclidean distance between the two vectors. If the distance is below a learned threshold, the signature is verified as genuine.
